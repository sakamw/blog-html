# Everyday Git 🛠️

A beginner-friendly tutorial blog built using only **HTML** to guide new developers through the basics of Git and version control. This project was created as part of a collaborative digital skills training program between **Teach2Give** and **Murang'a University of Technology**.

## 🌐 Live Demo

🔗 [Visit the blog here](https://sakamw.github.io/blog-html/)

---

## 📘 What You'll Learn

This static blog is a step-by-step guide covering:

1. Setting up your development environment
1. What Git is and why it matters
1. Installing Git on your computer
1. Configuring your name and email
1. Initializing a Git repository
1. Staging files and understanding the staging area
1. Committing changes with good commit messages
1. Working with branches (create, switch, merge, delete)
1. Viewing Git history
1. Next steps in your Git journey

Every section includes **clear explanations** and **command screenshots** to help readers follow along easily.
